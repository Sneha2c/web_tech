🎨 CSS Techniques Used

CSS Type Priority Where Used Examples
🔴 Inline CSS Highest (1000) Inside style attribute Colors, borders, image styling, button colors
🟢 Internal CSS Medium (100) <style> tag in head Glass cards, gradients, animations, hover effects
🔵 External CSS Lowest (1) style.css file Body background, base styles, font families

💡 CSS Specificity Rule: Inline > Internal > External

Inline CSS overrides all, perfect for unique elements. Internal CSS for page-specific styles. External CSS for consistent styling across multiple pages.
